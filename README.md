Wanderlust 🌍

A full-stack travel listing web application built with the MERN stack, where users can explore destinations, create and manage their own listings, and leave reviews.

🚀 Features:
🔐 User authentication (signup/login/logout)
🏡 Create, view, edit, and delete travel listings
⭐ Add and view reviews for listings
🖼️ Image upload for listings
🔍 Browse and search destinations
📱 Responsive design across devices
🛠️ Tech Stack

Frontend: React.js, HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB, Mongoose
Authentication: (e.g., Passport.js / JWT — update as per your project)
Other Tools: (e.g., Cloudinary for image storage, Bootstrap/Tailwind for styling — update as needed)


⚙️ Installation & Setup
1. Clone the repository <br/>
bash:
   git clone https://github.com/MahiJoshi27/WanderLust.git
   cd WanderLust
2. Install dependencies (backend) <br/>
bash:
   npm install
3. Install dependencies (frontend, if in separate folder) <br/>
bash:
   cd client
   npm install
4. Create a .env file in the root directory and add your environment variables <br/>
   MONGO_URI=your_mongodb_connection_string
   PORT=5000
5. Run the app <br/>
bash:
   npm start
